每日胶囊 · Daily Digest Capsule iOS / iPadOS SwiftUI

开发者中心

每日胶囊是一款结合资讯、效率与灵感的轻量级信息流服务。我们专注于为用户提供可定制化的每日要闻、待办提醒以及创意灵感板块， 并通过模块化架构快速迭代内容。此页面为开发者提供项目概览、技术栈说明、迭代节奏与合作方式。

产品愿景

聚焦核心信息

通过多渠道 API 聚合每日最重要的资讯内容，让用户在 3 分钟内完成信息摄取。

可扩展模块

采用 Capsule 模块模型，支持新闻、历史上的今天、名言、星座、天气等扩展内容。

原生体验

基于 SwiftUI + Combine 构建，轻量且易维护，支持灵动岛、Widget 等生态能力。

技术与架构

  • 使用 SwiftUI 构建 MVVM 架构，配合 modular 服务层确保可测试性与解耦。
  • 远程数据采用 Repository + RemoteDataSource 设计，支持 API 缓存与离线策略。
  • 集成 App Store 内购、AdMob App Open Ad 与 Banner 广告加载。
  • 统一的调试日志采用 `DebugLogger`，确保调试信息与生产日志隔离。

版本规划
